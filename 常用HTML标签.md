# 常用标签

1. 背景高亮

```html
<span style="background-color: yellow; font-size:28px; font-weight: bold;">请勿修改格式</span>
```

2. 视频标签

```html
<video src="https://gitee.com/HavertzPlatform/worker-picgo/raw/master/2025-06-26 18-02-46.mp4"></video>

<video 
  src="https://gitee.com/HavertzPlatform/worker-picgo/raw/master/2026-01-07%2017-53-49.mp4" 
  autoplay 
  muted 
  loop 
  controls
>
  您的浏览器不支持HTML5视频
</video>
```

3. 下载标签

```html
<a href="/path/to/file.pdf" download="filename.pdf">下载PDF文件</a>
```

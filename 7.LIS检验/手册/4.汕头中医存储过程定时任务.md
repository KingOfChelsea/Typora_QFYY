# 汕头中医存储过程定时任务

背景：LIS回传数据（数据+pdf报告），只上传了数据，报告延迟生成后，导致医生站无法看到报告数据，需要使用脚本，更新其状态，让他重新上传！

`临床反应有问题直接执行以下脚本删除`

```sql
EXEC DBMS_SCHEDULER.DROP_JOB('UPDATE_LAS_SAP_SAMPLEREG_JOB'); -- 删除定时任务
```

`从新创建定时任务`

```sql
BEGIN
  DBMS_SCHEDULER.CREATE_JOB (
    job_name        => 'UPDATE_LAS_SAP_SAMPLEREG_JOB',
    job_type        => 'STORED_PROCEDURE',
    job_action      => 'UPDATE_LAS_SAP_SAMPLEREG_PROC',
    start_date      => SYSTIMESTAMP,
    repeat_interval => 'FREQ=MINUTELY; INTERVAL=10',  
    enabled         => TRUE
  );
END;
/
```

`原脚本`

```sql
CREATE OR REPLACE PROCEDURE UPDATE_LAS_SAP_SAMPLEREG_PROC AS
  v_today_str VARCHAR2(8);  -- 使用字符串变量
  v_count NUMBER;
  v_error_msg VARCHAR2(4000);
BEGIN
  v_today_str := TO_CHAR(SYSDATE, 'YYYYMMDD');
  DBMS_OUTPUT.PUT_LINE('执行日期: ' || v_today_str);
  DBMS_OUTPUT.PUT_LINE('当前时间: ' || TO_CHAR(SYSDATE, 'YYYY-MM-DD HH24:MI:SS'));
  SELECT COUNT(*)
  INTO v_count
  FROM las_sap_samplereg t
  WHERE substr(t.lsptestform, 2, 1) = '2' 
  AND t.testdate >= v_today_str;
  DBMS_OUTPUT.PUT_LINE('找到 ' || v_count || ' 条符合条件的记录');
  IF v_count = 0 THEN
    DBMS_OUTPUT.PUT_LINE('没有符合条件的记录，无需更新');
    RETURN;
  END IF;
  UPDATE las_sap_samplereg t
  SET t.lsptestform = substr(t.lsptestform, 1, 1) || '0' || substr(t.lsptestform, 3)
  WHERE substr(t.lsptestform, 2, 1) = '2' 
  AND t.testdate >= v_today_str;  
  v_count := SQL%ROWCOUNT;
  DBMS_OUTPUT.PUT_LINE('成功更新了 ' || v_count || ' 条记录');
  COMMIT;
  DBMS_OUTPUT.PUT_LINE('更新完成，事务已提交');
  
EXCEPTION
  WHEN OTHERS THEN
    v_error_msg := SQLERRM;
    DBMS_OUTPUT.PUT_LINE('错误发生!');
    DBMS_OUTPUT.PUT_LINE('错误代码: ' || SQLCODE);
    DBMS_OUTPUT.PUT_LINE('错误信息: ' || v_error_msg);
    DBMS_OUTPUT.PUT_LINE('回滚事务...');
    ROLLBACK;
    RAISE;
END;
/
```

# 互认报告维护

互认报告主要分为两种形式，一种有检验科提供哪些仪器，哪些项目需要维护

## 1.维护项目类型

需要提供如下数据：

提供**仪器编码**,如下界面可以找到，如**图1**所示

![image-20260527113632493](https://gitee.com/HavertzPlatform/worker-picgo/raw/master/image-20260527113632493.png)

<center>图1</center>

提供项目编码，如图2

![image-20260527114125897](https://gitee.com/HavertzPlatform/worker-picgo/raw/master/image-20260527114125897.png)

<center>图2</center>

| 项目名称       | 项目简码 | 专业组 | 仪器  | 对应项目编码                                   |
| -------------- | -------- | ------ | ----- | ---------------------------------------------- |
| 粪便常规       | FBCG     | 临检   | FA280 | (S-COLOUR/S-APPE/S-RBC/S-WBC/XKLDJJ/Ph-fug/ZFQ |
| 粪寄生虫卵镜检 | FPE      | 临检   | FA280 | S-PARO                                         |
| 粪原虫镜检     | FPrM     | 临检   | FA280 | JSC                                            |
| 粪寄生虫镜检   | FPM      | 临检   | FA280 | JSC                                            |

`执行下方sql查询需要的仪器中的项目`

```sql
select a.*,a.rowid from las_com_machineitem  a 
where a.machineid like '%XN1500_1%' -- 仪器名称
and a.itemid = 'WBC' -- 项目名称
```

`执行下方语句进行更新`

```sql
update las_com_machineitem t set t.itemname = '*'||t.itemname 
where t.itemid in ()  -- 项目编码
and t.machineid = ''  -- 仪器编码
```

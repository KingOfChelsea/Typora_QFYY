# 体检接口文档

---

## 一、条码信息

### 调用方法

`http://10.161.211.42/LisWeb.asmx?op=AddTJBarcodeInfo` (具体现场具体对接)

### 说明
每个条码组成 `SapList` 传入，详见下方样例。

### 请求参数

| 字段           | 类型          | 名称                     | 备注                |
| -------------- | ------------- | ------------------------ | ------------------- |
| Barcode        | Varchar2(16)  | 条码号                   | 10位条码必须为8开头 |
| PatientID      | Varchar2(20)  | 体检号（患者唯一码）     | 患者唯一码          |
| PatientSEQ     | Varchar2(20)  | 体检流水号（就诊流水号） | 本次就诊唯一码      |
| PatientName    | Varchar2(50)  | 姓名                     |                     |
| PatientSex     | Varchar2(10)  | 性别                     | 男；女              |
| PatientAge     | Varchar2(10)  | 年龄                     | 数字                |
| PatientAgeUnit | Varchar2(10)  | 年龄单位                 | 岁，月，周，日等    |
| HisItemCode    | Varchar2(50)  | 申请项目编码             |                     |
| HisItemName    | Varchar2(100) | 申请项目名称             |                     |
| SampleType     | Varchar2(50)  | 样本类型                 | 全血，血清等        |
| Price          | Varchar2(10)  | 价格                     |                     |
| OrderDate      | Datetime      | 申请时间                 |                     |
| ExecSqn        | Varchar2(20)  | 执行单号                 | 本条医嘱的唯一码    |
| DoctorID       | Varchar2(20)  | 申请医生ID               |                     |
| DoctorName     | Varchar2(50)  | 申请医生姓名             |                     |
| NurseID        | Varchar2(20)  | 采样护士ID               |                     |
| NurseName      | Varchar2(50)  | 采样护士姓名             |                     |
| Factory        | Varchar2(100) | 地址                     |                     |
| Idenno         | Varchar2(20)  | 身份证号                 |                     |
| PatientTel     | Varchar2(20)  | 联系方式                 |                     |
| PatientID2     | Varchar2(20)  | 体检流水号               |                     |
| Birthday       | Datetime      | 出生日期                 |                     |
| DeptID         | Varchar2(20)  | 申请科室编码             |                     |
| DeptName       | Varchar2(50)  | 申请科室名称             |                     |
| ExecDeptID     | Varchar2(20)  | 执行科室编码             |                     |
| ExecDeptName   | Varchar2(50)  | 执行科室名称             |                     |
| SampleTime     | Varchar2(20)  | 采集时间                 | 2026-01-01 13:23:23 |
| diagnosis      | Varchar2(20)  | 诊断ID                   |                     |
| diagnosisid    | Varchar2(50)  | 诊断                     |                     |
| tubetype       | Varchar2(20)  | 容器颜色                 |                     |
| barcodememo    | Varchar2(20)  | 条码备注                 |                     |
| patientinfo    | Varchar2(20)  | 患者信息备注             |                     |
| hospitalid     | Varchar2(20)  | 院区编码                 |                     |

### 请求样例

```json
{
  "SapList": [
    {
      "Barcode": "8000504066",
```

### 返回参数

| 字段       | 类型         | 名称 | 备注               |
| ---------- | ------------ | ---- | ------------------ |
| ReturnCode | Varchar2(10) | 编码 | AA：成功，AE：失败 |
| ReturnMsg  | Varchar2(50) | 说明 |                    |

### 返回样例

```json
{
  "ReturnCode": "AE",
  "ReturnMsg": "条码号[800000001]，项目[F000002]已存在。"
}
```

## 二、条码作废

### 调用方法

`http://XXX.XXX.XXX.XXX:XXXX/LisWeb.asmx?op=DelTJBarcodeInfo`

### 请求参数

| 字段        | 类型         | 名称     | 备注 |
| ----------- | ------------ | -------- | ---- |
| Barcode     | Varchar2(20) | 条码号   |      |
| PatientID   | Varchar2(20) | 体检编号 |      |
| HisItemCode | Varchar2(50) | 项目编码 |      |

### 请求样例

#### 作废单个项目

```json
{
  "Barcode": "80000000001",
  "PatientID": "tj00000001",
  "HisItemCode": "F000000001"
}
```

#### 作废整个条码

```json
{"Barcode":"80000000001","PatientID":"tj00000001","HisItemCode":"ALL"}
```

### 返回参数

| 字段       | 类型         | 名称 | 备注               |
| ---------- | ------------ | ---- | ------------------ |
| ReturnCode | Varchar2(10) | 编码 | AA：成功，AE：失败 |
| ReturnMsg  | Varchar2(50) | 说明 |                    |

### 返回样例

```json
{"ReturnCode":"AA","ReturnMsg":"条码[800000001]作废成功"}
```

## 三、更新采集时间

### 调用方法

`http://XXX.XXX.XXX.XXX:XXXX/LisWeb.asmx?op=UpdateSample`

### 请求参数

| 字段       | 类型         | 名称       | 备注                |
| ---------- | ------------ | ---------- | ------------------- |
| Barcode    | Varchar2(20) | 条码号     |                     |
| Sampletime | Varchar2(20) | 采集时间   | YYYY-MM-dd hh:mm:ss |
| NurseID    | Varchar2(10) | 采集人工号 |                     |
| NurseName  | Varchar2(20) | 采集人姓名 |                     |

### 请求样例

```json
{
  "Barcode": "800000001",
  "Sampletime": "2025-01-01 12:29:01",
  "NurseID": "1001",
  "NurseName": "张三"
}
```

### 返回参数

| 字段       | 类型         | 名称 | 备注               |
| ---------- | ------------ | ---- | ------------------ |
| ReturnCode | Varchar2(10) | 编码 | AA：成功，AE：失败 |
| ReturnMsg  | Varchar2(50) | 说明 |                    |

### 返回样例

```json
{"ReturnCode":"AA","ReturnMsg":"数据更新成功"}
```

# 调用惠每接口实例

## 1. POST获取Token接口

### 1.1 接口地址

请求地址`http://10.161.11.42/cdss/api/outer/dec/ds/accredit/token`

### 1.2 Postman 配置

- 选择 **POST **切换到 **Body → `raw`**

```json
{
  "appKey": "HCj7x7oySKMMFWfH",
  "appSecret": "5dbd1cb1efdc68d364d78743834b4406f44a08f5",
  "name": "护理端获取VTE评估结果"
}
```

### 1.3 配置Header

在 **Headers** 中添加：

| Key          | Value            |
| ------------ | ---------------- |
| Content-Type | application/json |

## 2.使用 Token 调用 VTE 评估结果接口

### 2.1 地址 

请求地址：`http://10.161.11.42/cdss/api/outer/dec/ds/query/VteResult`

### 2.2  Postman 配置

在 **Body → raw → JSON** 中传入

```json
{
  "patient_guid": "0000514603",
  "serial_number": "ZY0010000514603"
}
```

### 2.3 配置Header 

在 **Headers** 中添加：

- `DS-Authorization`：值为刚刚获取的 `access_token`
- `Content-Type`：`application/json`

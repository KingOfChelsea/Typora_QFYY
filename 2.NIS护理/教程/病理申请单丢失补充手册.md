# 病理申请单丢失补充手册

## 1.1通过患者门诊流水号/住院流水号查出申请单

- 查出丢失的申请单中的`combo_no`，如**图1**

```sql
select r.mo_date,r.combo_no,r.* from met_ordo_order r
where r.clinic_no = '5432858'
  and r.order_name like '%病理申请单%' 
```

## 2.补充结构化数据

```sql
-- 祈福医院现场 直接执行下面的sql
select a.*,rowid from his_fshv1.met_orde_applysheet_bl  a
where a.order_guid in ('90970507') -- 组合号
```

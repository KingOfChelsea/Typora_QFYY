# 血压待测单排查报告

1. 频次编码不能开立为，BIDCS（每日2次(早、晚)），必须为BID、Q12H、Q2H、 Q3H、 Q4H 、Q6H 、Q8H QD 、TID 、QH

**待测单查询sql**

```sql
select  a.frequency_code,a.frequency_name,(select o.name from fin_ipr_inmaininfo o where o.inpatient_no = a.inpatient_no  )as name, a.* from met_ipm_order a --医嘱资料档
where 1=1 -- a.mark6 = '66704729'
and a.nurse_cell_code = '42800'
and a.item_name like '%血压%' 
order by a.MO_DATE desc

SELECT p.* FROM NIS_ORDI_PLAN p
 where p.plan_item_id = 'FourTimesBPTask'
   and p.inpatient_code = 'ZY0020000514042'
	 order by p.create_date desc;

SELECT *FROM NIS_ORDI_PLAN_item pp where pp.parent_id in(SELECT id FROM NIS_ORDI_PLAN p
 where p.plan_item_id = 'FourTimesBPTask'
   and p.inpatient_code = 'ZY0020000514042' )
	 order by pp.time desc;--任务子表
	 
	 

select a.inpatient_no,i.name ,a.term_id,a.term_name,a.order_name from met_ordi_order a
LEFT JOIN fin_ipr_inmaininfo i ON  a.inpatient_no = i.inpatient_no
where a.order_name like '%血压%' 
--and a.term_id = 'M' 
and a.nurse_cell_code = '42800' 
and a.term_name = a.order_name
and i.name like '%程方石%'
```

# 祈福医院护理会诊维护

## 1.维护护理专科小组

1. 打开护士长界面（需要护士长权限），打开如下界面

   ![image-20250725083021906](https://gitee.com/HavertzPlatform/worker-picgo/raw/master/image-20250725083021906.png)

2. 维护护理组

   - [x] 添加新建对应的护理组，并添加对应的护理组

   ![](https://gitee.com/HavertzPlatform/worker-picgo/raw/master/20250624181242875.png)

   - [x] 添加会诊组

   ![image-20250624181805303](https://gitee.com/HavertzPlatform/worker-picgo/raw/master/image-20250624181805303.png)

   - [x] 添加对应的人员

   ![image-20250624181820036](https://gitee.com/HavertzPlatform/worker-picgo/raw/master/image-20250624181820036.png)

   ## 2.发起护理会诊

   - 双击患者列表

   ![image-20250725084743702](https://gitee.com/HavertzPlatform/worker-picgo/raw/master/image-20250725084743702.png)

   - 双击患者姓名，进入NIS工作站

   ![image-20250725084950064](https://gitee.com/HavertzPlatform/worker-picgo/raw/master/image-20250725084950064.png)

   - 双击会诊申请对此患者发起会诊申请

 ![image-20250725085255858](https://gitee.com/HavertzPlatform/worker-picgo/raw/master/image-20250725085255858.png)

- 点击添加申请

![image-20250725085938563](https://gitee.com/HavertzPlatform/worker-picgo/raw/master/image-20250725085938563.png)

- 点击提交

![image-20250725090105479](https://gitee.com/HavertzPlatform/worker-picgo/raw/master/image-20250725090105479.png)

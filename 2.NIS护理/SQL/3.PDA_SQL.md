## 1. PDA移动护理医嘱列表

```sql
SELECT 
	e.* 
FROM 
    view_met_nui_exerecord e
WHERE 1=1 
	AND e.valid_flag = '1'  -- 只查询有效记录  
    AND e.inpatient_no LIKE '%501161'  -- 住院流水号
ORDER BY 
    CASE WHEN e.exe_state = '0' THEN 0 WHEN e.exe_state = '1' THEN 1 ELSE 2 END,  -- 待执行排最前
    e.use_time ASC, 
    e.exec_sqn;
```

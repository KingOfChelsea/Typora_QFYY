# 上线查询数据常用SQL

##  1.查询所有表

- 查询HIS系统中所以得表
  ```sql
          -- 0.查询HIS系统中所以得表格1
          select t.OWNER, t.TABLE_NAME, tc.COMMENTS, t.NUM_ROWS
            from dba_tables t
            left join dba_tab_comments tc
              on t.OWNER = tc.OWNER
             and t.TABLE_NAME = tc.TABLE_NAME
           where exists (select 1
                    from dba_users d
                   where d.CREATED > to_date('2025-01-01', 'yyyy-mm-dd')
                     and d.USERNAME = t.OWNER)
             and tc.COMMENTS like '%婴儿%'
           order by t.OWNER,
                    t.TABLE_NAME
  ```

# 门诊业务SQL

## 1.查门诊终端确认

```sql
 SELECT a.*, a.ROWID  FROM met_tec_ta_detail a  where a.card_code = '0002503751' and a.clinic_code = '4715480'; -- 终端确认业务子表 
 select t.* from fin_opb_feedetail  t where t.card_no = '0002503751'and t.clinic_code = '4715480'; --  处方明细表 
 SELECT a.*, a.ROWID  FROM met_nui_cancelitem a where a.inpatient_no = '4715480' -- 终端退费申请表
```

## 2.挂号主表

```sql
select rowid, t.see_date,t.pay_mode,t.pay_cost,t.doct_name , t.reg_date,t.oper_code,t.ynbook, t.ynsee, t.seeno,t.doct_code, t.dept_name,t.see_docd,t.* from fin_opr_register t --挂号信息表
where 1=1
-- and t.idenno = '1234556'
and t.name like '%涂梓健%'
-- and t.clinic_code = '1393626' 
order by t.reg_date desc;
```

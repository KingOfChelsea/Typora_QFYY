# VTE风险评估对接视图手册

## 1. 视图说明
本视图用于获取VTE（静脉血栓栓塞症）风险评估结果，包括风险等级、分数等关键信息。

## 2. 视图结构

### 视图名称：`v_vte_assessment_results`

### 字段说明：

| 字段名          | 数据类型     | 说明                           | 是否主键 |
| --------------- | ------------ | ------------------------------ | -------- |
| assessment_id   | VARCHAR(50)  | 评估唯一标识                   | 是       |
| inpatient_no    | VARCHAR(30)  | 住院流水号                     | 否       |
| patient_id      | VARCHAR(20)  | 患者                           | 否       |
| name            | VARCHAR(50)  | 患者姓名                       | 否       |
| dept_name       | VARCHAR(100) | 科室名称                       | 否       |
| bed_no          | VARCHAR(20)  | 床位号                         | 否       |
| assessment_time | DATETIME     | 评估时间                       | 否       |
| assessment_type | VARCHAR(20)  | 评估类型（Caprini/Padua等）    | 否       |
| total_score     | INT          | 总分                           | 否       |
| risk_level      | VARCHAR(20)  | 风险等级                       | 否       |
| is_completed    | TINYINT(1)   | 是否完成（0-未完成，1-已完成） | 否       |
| assessor_id     | VARCHAR(20)  | 评估人ID                       | 否       |
| assessor_name   | VARCHAR(50)  | 评估人姓名                     | 否       |
| create_time     | DATETIME     | 创建时间                       | 否       |
| update_time     | DATETIME     | 更新时间                       | 否       |

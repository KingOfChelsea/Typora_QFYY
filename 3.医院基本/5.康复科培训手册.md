# 一.HIS费用类

## 1.办理出院节点

### 1.1 未终端确认

概念：`未终端确认`，就是一项检查/检验的费用，还在‘路上’，没有成功‘落’到病人账上。

- **开立医嘱：** 由病区医生为病人开了一张CT检查单。

- **审核执行：** 护士核对无误后执行了这条医嘱，检查申请发出。

- **终端确认：** 由**终端科室——这里是CT室**，在系统中点击“确认”。只有他们“签收”了，这个“订单”才算最终完成，费用才能从病人账户划扣。

如`图1-1`所示，需要办理出院,检查检验项目部分确实`不需要做`，则需要医生`直接作废`对应的检查检验的医嘱，如`已经做了`则需要对应的终端科室进行`确认`！

<center>图1-1</center>
